# Pill-dispenser 
Arduino project for a medication reminder with:
- I2C LCD
- 4x4 keypad
- Buzzer + LED
- Servo motor for dispensing

## Features
- Set time using keypad (`*`)
- Start countdown with `#`
- Audible and visual alarm when time reaches zero
- Servo moves to simulate dispensing

## Hardware
- Arduino Uno
- 16x2 I2C LCD (address 0x20 in Tinkercad, 0x20/0x27/etc. in Wokwi)
- 4x4 membrane keypad
- Buzzer + LED + resistor
- Servo
